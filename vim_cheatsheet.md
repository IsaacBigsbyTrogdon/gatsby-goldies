# nvim Cheatsheat

## vim notation

```
<S-…>		shift-key			*shift* *<S-*
<C-…>		control-key			*control* *ctrl* *<C-*
<M-…>		alt-key or meta-key		*META* *ALT* *<M-*
<A-…>		same as <M-…>			*<A-*
<D-…>		command-key or "super" key	*<D-*
```

## vim General

```
:m 12 # move to after line 12.
yyp # Duplicate a line.
:bd # Close buffer/file.
:hs / :vs # new horizontal or vertical split.
<C-ww> # switch panes/splits.
<C-w>c # close split.

    CTRL+W,+ enlarges the current split panel
    CTRL+W,- shrinks the current split panel
    CTRL+W,= evens the split panels size
    CTRL+W,CTRL+W moves the focus to the next split panel

> # indent highlighted text (Visual Mode)
```
